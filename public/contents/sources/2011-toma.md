# Resolving the Cycle Skip Introduced by the Multi-Layer Static Model using a Hybrid Approach

**Summary**: The thesis investigates the problem of artificial cycle skips in seismic data caused by multi-layer static models and proposes a hybrid approach. By combining high-frequency components from single-layer models with low-frequency components from multi-layer models to create an external pilot, the study successfully resolves these skips while maintaining regional consistency.

---

## Research Problem

Multi-layer static models often introduce artificial cycle skips (data breaks) in seismic data due to horizontal ray paths interacting with velocity inversions or buried anomalies. These skips are frequently irresolvable using conventional residual static correction techniques.

## Objectives

- Identify the causes of cycle skips introduced by multi-layer static models.
- Develop a hybrid approach combining single-layer and multi-layer static components to create an external pilot for residual statics calculation.
- Evaluate the effectiveness of the hybrid approach in resolving cycle skips compared to conventional methods.

## Methods

The study utilizes a two-stage workflow. Stage 1 involves creating a data-driven external trace volume pilot by applying high-frequency static components from a single-layer model to pre-stack data, followed by one pass of residual statics. Stage 2 involves applying low-frequency statics from the multi-layer model to the post-stack data (which is first conditioned with dip coherency enhancement, band-pass filtering [12-62 Hz], and time-variant automatic trace scaling) to serve as an external pilot for a final residual statics calculation. The results are compared against conventional methods using both pre-stack shot records and post-stack cross-line/in-line sections.

## Data and Materials

A 3D seismic data set from the Haba Field in Eastern Saudi Arabia, including pre-stack gathers, post-stack time slices (at 700 ms and 1400 ms), and cross-line/in-line profiles.

## Key Findings

- The multi-layer model is more realistic than the single-layer model as it represents velocity in weathering and sub-weathering layers, but it can introduce cycle skips due to horizontal ray paths. ([[2011-toma|Toma, 2011, p. 56]])
- Cycle skips are often more apparent in cross-line directions compared to in-line directions, which can lead to misleading interpretations if only in-lines are examined. ([[2011-toma|Toma, 2011, p. 57]])
- The hybrid approach successfully resolved cycle skips in the pre-stack shot records when compared to conventional multi-layer static solutions. ([[2011-toma|Toma, 2011, p. 60]])
- Post-stack time slices at 700 ms and 1400 ms showed significantly improved continuity using the hybrid approach, eliminating artifacts seen in the conventional method. ([[2011-toma|Toma, 2011, p. 63]])
- The difference between multi-layer and single-layer low frequency statics in the study area was significant, ranging from -11 to +120 ms. ([[2011-toma|Toma, 2011, p. 43]])

## Recommendations and Future Work

The single-layer static model should be used alongside the multi-layer model as a quality control measure. Standard internal pilot residual statics techniques should be attempted first before applying the proposed hybrid approach.
