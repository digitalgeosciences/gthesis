## Key Findings From This Location

- Java Island is one of the most densely populated regions in Indonesia, with 141 million people (in 2012). ([[2015-latif|Latif, 2015, p. 34]])

- Merapi is an active stratovolcano situated in the middle of Java Island (7.54°S, 110.446°E). ([[2022-argadestya|Argadestya, 2022, p. 26]])

- The West Java province holds the highest population in the country (49,565,200). ([[2025-rachmadan|Rachmadan, 2025, p. 23]])
