# Tomographic Inversion

**Summary**: A method to image 3D velocity structures in the Earth's interior using travel-time residuals from seismic waves.

---

## How It Is Used in the Thesis Collection

- [[2025-kesumastuti|Kesumastuti, 2025]] used tomographic inversion via the FMTOMO package to create a 3D velocity model of the lithosphere and upper mantle.
