# Carreau Model

**Summary**: A non-linear model used to describe the viscosity of a fluid as a function of shear rate.

---

## How It Is Used in the Thesis Collection

- [[2012-omer|Omer, 2012]] applied Carreau model via non-linear regression to determine zero-shear rate viscosity for various PAM concentrations.
