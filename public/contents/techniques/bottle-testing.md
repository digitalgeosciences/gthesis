# Bottle Testing

**Summary**: A non-destructive method to evaluate gel strength and gelation rate by assigning a letter code based on flow behavior.

---

## How It Is Used in the Thesis Collection

- [[1997-al-nashmi|Al-Nashmi, 1997]] used bottle testing to screen gel formulations rapidly and inexpensively across various temperatures and concentrations.
