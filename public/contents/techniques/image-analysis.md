# Image Analysis

**Summary**: Digital processing of petrographic images to quantify and characterize pore geometry.

---

## How It Is Used in the Thesis Collection

- [[2014-mohammed|Mohammed, 2014]] used Petrographic Image Analysis (PIA) to identify the pore sizes, aspect ratio and roundness factor for each pore type.

- [[2016-siddiqui|Siddiqui, 2016]] used ImageJ image analysis software to evaluate foam texture from captured images.
