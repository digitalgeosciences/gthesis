# Nonlinear Multiple Regression

**Summary**: A statistical method used to estimate the parameters of a nonlinear model using least-squares or weighted least-squares.

---

## How It Is Used in the Thesis Collection

- [[1994-al-fattah|Al-Fattah, 1994]] used nonlinear multiple regression via the SAS NLIN procedure to develop empirical equations for relative permeability.
