# Linear Regression

**Summary**: A statistical method used to estimate the relationship between variables by fitting a line or curve to data points.

---

## How It Is Used in the Thesis Collection

- [[2014-kola|Kola, 2014]] used linear regression to estimate elastic parameter contrasts from TPS amplitudes.

- [[2018-putra|Putra, 2018]] used linear-regression to align arrival times (t_obs) in real seismic data to calculate ideal trace energy.

- [[1994-al-fattah|Al-Fattah, 1994]] used linear multiple regression to determine coefficients for new empirical equations.

- [[1997-al-gahtani|Al-Gahtani, 1997]] applied linear regression to determine coefficients for completion efficiency and optimum well length correlations.

## Key Results and Findings

- Estimation of $\beta/\alpha$ and $\Delta\beta/\beta$ were recovered better than $\Delta\rho/\rho$ using linear regression on TPS amplitudes. ([[2014-kola|Kola, 2014, p. 77]])

- Used to determine the shift from t_obs to t_shift to compare real data against an idealized scenario. ([[2018-putra|Putra, 2018, p. 68]])

- The resulting models were evaluated against published equations, showing superior performance in predicting water-oil relative permeability. ([[1994-al-fattah|Al-Fattah, 1994, p. 16]])

- Correlation coefficients were obtained using non-linear regression analysis to relate partially perforated wells to open-hole performance. ([[1997-al-gahtani|Al-Gahtani, 1997, p. 48]])
