# Co Kriging

**Summary**: A geostatistical estimation method that uses multiple correlated variables to improve the estimate of a primary variable.

---

## How It Is Used in the Thesis Collection

- [[1991-makkawi|Makkawi, 1991]] used co-kriging to improve MgO% estimates by incorporating correlated CaO% data.
