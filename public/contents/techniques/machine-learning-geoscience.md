## Key Results and Findings

- RBF neural networks showed improved correlation and lower error compared to multi-linear regression. ([[2015-fuad|Fuad, 2015, p. 64]])

- Used ANN to predict permeability and classify lithology with high correlation coefficients. ([[2020-abdallah|Abdallah, 2020, p. 70]])

- DBSCAN is preferred in most cases because it does not require a training data set compared to supervised methods. ([[2021-almashhor|AlMashhor, 2021, p. 34]])

- The CNN achieved 100% accuracy on training/validation data and outperformed traditional methods like STA/LTA. ([[2021-shaheen|Shaheen, 2021, p. 41]])

- Unsupervised methods do not require labeled data, making them more flexible for various geophysical tasks. ([[2022-awais|Awais, 2022, p. 36]])

- Machine learning offers potential for detecting, locating, and interpreting seismic events using various architectures like FNOs. ([[2023-suleymanli|Suleymanli, 2023, p. 21]])

- Consecutive neural networks provided a 10x speedup over traditional and other tested models. ([[2023-almarzooq|Almarzooq, 2023, p. 16]])

- Machine learning has seen exponential growth in the last decade for tasks like image classification and object detection. ([[2025-ferreira|Ferreira, 2025, p. 17]])
