# Thomsen Parameters

**Summary**: Parameters used to describe the elastic properties and anisotropy of a transversely isotropic medium.

---

## How It Appears in the Thesis Collection

- [[2024-makki|Makki, 2024]] calculated Thomsen parameters to characterize the TTI nature of the sand dune.

## Scientific Context

- [[2024-makki|Makki, 2024]] Thomsen parameters quantify how much the P-wave velocity varies with angle relative to the symmetry axis.

## Measurement and Characterization Methods

- [[2024-makki|Makki, 2024]] Determined using a non-linear least squares algorithm fitting the measured data to a TTI function.

## Key Findings Across Sources

- The values were epsilon = 0.03461668 and delta = 0.00511634, confirming a weak anisotropic medium. ([[2024-makki|Makki, 2024, p. 73]])
