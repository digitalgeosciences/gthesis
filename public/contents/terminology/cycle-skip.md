# Cycle Skip

**Summary**: A discontinuity in seismic data where the signal is shifted out of phase, often caused by incorrect static corrections.

---

## How It Appears in the Thesis Collection

- [[2011-toma|Toma, 2011]] analyzed cycle skips resulting from multi-layer model applications.

- [[2018-mohamed|Mohamed, 2018]] addressed the problem of cycle skipping by using SWI to improve initial velocity models.

## Scientific Context

- [[2011-toma|Toma, 2011]] Cycle skips are a major problem in seismic interpretation as they create false structures.

- [[2018-mohamed|Mohamed, 2018]] addressed the problem of cycle skipping by using SWI to improve initial velocity models.

## Measurement and Characterization Methods

- [[2011-toma|Toma, 2011]] identified through pre-stack and post-stack data analysis.

## Key Findings Across Sources

- Artificial cycle skips can be misleading for interpreters and introduce false subsurface images. ([[2011-toma|Toma, 2011, p. 10]])

- SWI helped avoid the problem of cycle skipping, which leads to local minimum trapping and the wrong velocity model. ([[2018-mohamed|Mohamed, 2018, p. 15]])
