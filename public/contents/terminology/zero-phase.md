# Zero Phase

**Summary**: A seismic wavelet that is symmetrical with maximum amplitude at time zero.

---

## How It Appears in the Thesis Collection

- [[2023-bargees|Bargees, 2023]] evaluated zero-phase seismic data using the HEEH method to overcome limitations of standard pickers.

## Scientific Context

- [[2023-bargees|Bargees, 2023]] Commonly produced by Vibroseis sources, but often requires conversion for standard pickers.

## Measurement and Characterization Methods

- [[2023-bargees|Bargees, 2023]] Analyzed via Hilbert transform and envelope extraction.

## Key Findings Across Sources

- Zero-phase wavelets are captured as windows of ones by the empirical rule when processed via Hilbert transform. ([[2023-bargees|Bargees, 2023, p. 13]])
