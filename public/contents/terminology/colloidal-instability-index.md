# Colloidal Instability Index

**Summary**: An index used to identify asphaltene deposition problems based on SARA fractions.

---

## How It Appears in the Thesis Collection

- [[2023-suleiman|Suleiman, 2023]] used CII to screen crude samples for stability, where values ≥0.9 indicate instability.

## Scientific Context

- [[2023-suleiman|Suleiman, 2023]] Used as a quick tool for screening crude samples for potential deposition issues.

## Measurement and Characterization Methods

- [[2023-suleiman|Suleiman, 2023]] Calculated from SARA analysis data.

## Key Findings Across Sources

- For CII ≥0.9, the crude is unstable whereas, for CII ≤0.7, crude stability is certain. ([[2023-suleiman|Suleiman, 2023, p. 70]])
