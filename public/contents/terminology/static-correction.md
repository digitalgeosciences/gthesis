# Static Correction

**Summary**: The adjustment of seismic travel times to account for variations in the near surface.

## Key Findings Across Sources

- High frequency statics are applied before NMO, while low frequency statics are applied after NMO. ([[2011-toma|Toma, 2011, p. 39]])

- Static correction accounts for differences in elevation between sources and receivers or variations in the weathering layer. ([[2021-almashhor|AlMashhor, 2021, p. 17]])

- Correcting for 'time-in-sand' is essential before applying refraction tomography to solve for weathering layer statics. ([[2022-aljishi|AlJishi, 2022, p. 21]])

- Surface-consistent residual statics can be used as a proxy to identify incorrect NMO velocities. ([[2023-muhammad|Muhammad, 2023, p. 45]])

- Integrated models resolved statics better than models using only uphole data. ([[2002-al-ali|Al-Ali, 2002, p. 19]])
