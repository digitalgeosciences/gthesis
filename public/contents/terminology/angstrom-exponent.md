# Angstrom Exponent

**Summary**: A ratio of the extinction coefficients at different wavelengths used to determine particle size.

---

## How It Appears in the Thesis Collection

- [[2019-alotaibi|Alotaibi, 2019]] used Angstrom exponent to distinguish between fine and coarse mode aerosols.

## Scientific Context

- [[2019-alotaibi|Alotaibi, 2019]] Used to differentiate between natural dust and anthropogenic pollution.

## Measurement and Characterization Methods

- [[2019-alotaibi|Alotaibi, 2019]] Calculated from multi-wavelength AOD data.

## Key Findings Across Sources

- An AE value $\ge$ 1 indicates fine-mode particles (e.g., smoke), while < 1 indicates coarse-mode particles (e.g., dust). ([[2019-alotaibi|Alotaibi, 2019, p. 40]])

## Contrasting Values and Interpretations

- [[2019-alotaibi|Alotaibi, 2019]] Values above 1.0 indicate larger particles; values below 1.0 indicate smaller particles.
