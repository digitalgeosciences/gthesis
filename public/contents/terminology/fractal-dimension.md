# Fractal Dimension

**Summary**: A mathematical description of the complexity and heterogeneity of a pore system.

---

## How It Appears in the Thesis Collection

- [[2024-isah|Isah, 2024]] calculated fractal dimensions to quantify the complexity of micropores, mesop pores, and macropores.

- [[2024-jibrin|Jibrin, 2024]] determined the fractal dimension of each rock type using the box counting method.

## Scientific Context

- [[2024-isah|Isah, 2024]] Used to characterize complex pore systems in tight rocks where traditional Euclidean geometry is insufficient.

- [[2024-jibrin|Jibrin, 2024]] determined the fractal dimension of each rock type using the box counting method.

## Measurement and Characterization Methods

- [[2024-isah|Isah, 2024]] Derived from NMR T2 distribution data using the relationship between T2 and pore volume.

## Key Findings Across Sources

- Macropore D3 ranged from 2.41 to 2.96, indicating medium complexity. ([[2024-isah|Isah, 2024, p. 115]])


- Clay samples exhibited higher fractal dimensions (1.9 and 1.78) compared to feldspar (1.62 and 1.63). ([[2024-jibrin|Jibrin, 2024, p. 76]])
