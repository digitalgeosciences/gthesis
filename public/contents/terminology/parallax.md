# Parallax

**Summary**: The apparent change in the position of an object based on the angle of observation.

---

## How It Appears in the Thesis Collection

- [[2024-jahlan|Jahlan, 2024]] applied the parallax phenomenon from photography to identify and replace missing information in seismic data caused by dune occlusions.

## Scientific Context

- [[2024-jahlan|Jahlan, 2024]] Used as a conceptual bridge between image processing and seismic data to solve occlusion problems.

## Measurement and Characterization Methods

- [[2024-jahlan|Jahlan, 2024]] Analogy with multi-image photography.

## Key Findings Across Sources

- By using multiple viewpoints (CDPs), information hidden by a dune in one view can be retrieved from another. ([[2024-jahlan|Jahlan, 2024, p. 18]])
