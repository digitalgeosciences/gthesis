# Sulfate Scale

**Summary**: The precipitation of calcium sulfate minerals which can block pores in carbonate rocks.

---

## How It Appears in the Thesis Collection

- [[2013-ibrahim|Ibrahim, 2013]] investigated sulfate scale prevention using chelating agents during seawater injection.

## Scientific Context

- [[2013-ibrahim|Ibrahim, 2013]] Sulfate scale is a major concern when using seawater for pressure maintenance or EOR.

## Measurement and Characterization Methods

- [[2013-ibrahim|Ibrahim, 2013]] Measured via permeability changes after exposure to different brine types.

## Key Findings Across Sources

- Seawater injection caused 28% loss in core permeability due to sulfate scaling, which was reduced to 7% with proper dilution and chelation. ([[2013-ibrahim|Ibrahim, 2013, p. 84]])
