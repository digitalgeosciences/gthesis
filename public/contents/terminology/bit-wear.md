# Bit Wear

**Summary**: The degradation of the drill bit's cutting structure over time and distance.

---

## How It Appears in the Thesis Collection

- [[2014-ahmed|Ahmed, 2014]] monitored bit wear to determine optimal pull points for bit replacement.

- [[2022-al-dahmoush|Al-Dahmoush, 2022]] analyzed bit wear as a primary driver for MSE increases in abrasive sandstone.

## Scientific Context

- [[2014-ahmed|Ahmed, 2014]] Predicting wear allows for planned bit changes rather than reactive ones.

- [[2022-al-dahmoush|Al-Dahmoush, 2022]] analyzed bit wear as a primary driver for MSE increases in abrasive sandstone.

## Measurement and Characterization Methods

- [[2014-ahmed|Ahmed, 2014]] Calculation of Wc and tracking the Wf value over depth.

## Key Findings Across Sources

- Bit wear was modeled using a coefficient (Wc) to predict the Wf value, which determines when a bit should be pulled. ([[2014-ahmed|Ahmed, 2014, p. 88]])


- 90% of bits showed 'worn teeth' (WT) as the main dulling characteristic. ([[2022-al-dahmoush|Al-Dahmoush, 2022, p. 42]])
