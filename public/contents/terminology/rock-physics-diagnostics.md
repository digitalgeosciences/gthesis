# Rock Physics Diagnostics

**Summary**: The methodology used to establish a Rock Physics Model by analyzing well data.

---

## How It Appears in the Thesis Collection

- [[2020-almumtin|Almumtin, 2020]] used rock physics diagnostics (RPD) to determine the best-fit model for the Saudi clastic field.

## Scientific Context

- [[2020-almumtin|Almumtin, 2020]] It is the core process for selecting and refining the RPM based on actual well data.

## Measurement and Characterization Methods

- [[2020-almumtin|Almumtin, 2020]] Cross-plotting elastic variables (e.g., Vp) vs. volumetric variables (e.g., porosity).

## Key Findings Across Sources

- RPD involves cross-plotting an elastic variable against a volumetric variable and color-coding by a third variable. ([[2020-almumtin|Almumtin, 2020, p. 18]])
